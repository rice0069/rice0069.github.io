The 'strong' tag:
Used to display text in Bold.
The 'sup' tag:
Displays smaller text half a character above the normal line. Commonly used with dates (October 9th) and money (15$).
The 'em" tag:
Used to define Emphasize text, displaying it in Italics.
The 'mark' tag:
Used to Highlight text.
The 'hr' tag:
Used to create a horizontal line across the page.
